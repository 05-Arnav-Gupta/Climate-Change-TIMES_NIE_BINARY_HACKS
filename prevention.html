How can we prevent it?



We can ensure a few steps to ensure that the climate is protected. We cant stop it entirely but we can prevent a lot of it. Certain steps can help us to reduce it.


Replace Fossil fuels with more greener sources of energy like solar, wind, hydro etc. This will save a lot of fuel as well as prevent a lot of pollution caused due to burning them.


Public Pool can be used to reduce pollution caused due to vehicles. Using private vehicles for more than 1 person or using public transport can help in solving this problem


Afforestation can be done to increase the amount of trees which will cool the environment around it and bring rain.


Other methods like Recycling, using energy-efficient devices also work.
